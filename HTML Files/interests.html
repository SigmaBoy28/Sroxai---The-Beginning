After trying out ROBOTICS... I started to go in pursuit of building my own AI.
Why? Just for fun... I wanted to build JARVIS, but it isn't possible. Here's how my AI journey went:

1. A complete over-shot bot:
ChatGPT... I was just copy-pasting 90% of code from it. I added features that i haven't used till now!
Features like: file_opener, folder_opener, file_deleter, app_opener, website_opener, addition, subtraction, multiplication, division, exponents, factorials..... A lot of features.
Eventually, my code became like a Spaghetti
To the point where i didn't know what it was even capable of... I rage-quitted there.

2. ROAIS - first online bot
Second time... I named it "ROAIS". I wanted to buold it completely by myself, in fact at the start, i even coded a few basic functions.
But, soon it became copy-pasted bot. But this time, i added Gemini-Api to it. First time, it was fun.
Soon, i found out that it had daily-limits. Also, there was very high latency. I gave up and forgot about it to..

3. Roais - better than before This time, i coded around 50% of it myself. I added custom welcoming messages, weather API, ASCII art and other things.
This time i used HuggingFace API just to find it had more latency then Gemini...
I used it for a few days and it felt slower than manually doing things.
So, that's how i quit once again...

4. SCBS: offline bot It is my present bot that i completely programmed myself. It only does basic things like opening apps, websites, timers and other basic things.
I am proud of this cause i completelycreated it myself without seeing anywhere.
SCBS = "System commander bot - Sroxai"

THE END, I still have one goal for now: Build and train an evolving AI that runs completely local and offline for my PC.
I... don't know when to start and where to start, but i will do it.
